Welcome to KKU Campus Guide

This webpage introduces important places inside Khon Kaen University.

1. ประตูหลักมหาวิทยาลัย (Main Gate)

จุดแลนด์มาร์กที่นักศึกษามักใช้ถ่ายรูปและเป็นจุดเริ่มต้นของมหาวิทยาลัยขอนแก่น

2. หอสมุดกลาง (Central Library)

แหล่งรวมความรู้ หนังสือ และพื้นที่อ่านหนังสือที่เงียบสงบสำหรับนักศึกษา

3. สนามกีฬา 50 ปี

พื้นที่สำหรับออกกำลังกาย แข่งขันกีฬา และจัดกิจกรรมต่าง ๆ ภายในมหาวิทยาลัย

4. ศูนย์ประชุมอเนกประสงค์กาญจนาภิเษก

พื้นที่รวมชมรมและกิจกรรมต่าง ๆ สำหรับนิสิตนักศึกษาในการทำงานร่วมกัน
